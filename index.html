This website is more of an exercise than anything.

Right now, we're going to learn about lists.

  • This is an "undordered list"
  • You can tell because there are no numbers.
    • And also because of this forward-tabbed little guy over here.
    • This one too, and we're using the "\UL" to create the second tier of lists - right above the last line, and a tab to the left.
    • Each sentence is wrapped with a "\LI" too.

H3ere's a quick way to remember these codes:

  1. If you're looking for an ordered list (a list with numbers), then you're looking for the ol' "\OL"
  2. But if you're trying to get around an unordered part of town (dots), you need the ULtimate command of "\UL"
  3. And of course, there's Bruce Li. Always remember: If you're making a point, make it with "\LI"

"But what about if I'm writing my dissertation?"

Well, you're in luck today, kid.

  1. There's no better way to lipstick a pig than with Roman Numerals.
  2. It's like we're back in 2nd grade. Okay, here's the thing: it's just an i.
  3. Seriously. Just type "type=\i" with proper formatting into the ol' "\OL" and you're as golden as that switchblade wielding human-boy horse creature.
